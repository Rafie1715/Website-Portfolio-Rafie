LaMiMin (Lapar Miskin Minus) App Interface

A UI/UX design project focused on solving societal challenges.

Project Background

This application interface was designed to address two critical Sustainable Development Goals (SDGs): SDG 1 (No Poverty) and SDG 2 (Zero Hunger). We chose these themes because poverty and hunger are fundamental, interconnected issues. This project explores how a well-designed digital interface can facilitate campaigns, donations, and aid distribution to tackle these challenges.

Tools Used

Design Process

The design process focused on creating an intuitive and trustworthy user experience. Key considerations included making the donation process as simple as possible, providing clear information about ongoing campaigns, and ensuring transparency in fund distribution. We used Figma to create wireframes, high-fidelity mockups, and an interactive prototype to simulate the user flow from discovery to donation.
