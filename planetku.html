Planetku (Smart Waste Management App)

A capstone project for the Bangkit Academy 2024 program.

Project Background

Waste management is a significant challenge in many urban areas. The goal of Planetku was to create a mobile application that empowers users to manage their waste more effectively and sustainably. The app aims to solve problems like incorrect waste sorting, lack of awareness about recycling value, and difficulty in finding nearby waste banks.

My Role & Responsibilities

  • Developing the core features of the Android application using Kotlin.
  • Implementing the user interface based on the UI/UX design from the design team.
  • Integrating the mobile app with Firebase for user authentication and real-time database functionalities.
  • Connecting the app to a back-end API developed by the cloud computing team for features like price prediction.
  • Collaborating with the Machine Learning team to implement the smart waste classification feature using the device's camera.

Technology Stack

Challenges & Solutions

One of the biggest challenges was implementing the camera-based waste classification feature efficiently without slowing down the user's device. The initial model was quite large and slow.

Solution: We worked closely with the Machine Learning team to optimize the TensorFlow Lite model for mobile deployment. On the app side, I implemented asynchronous processing (`Coroutines` in Kotlin) to ensure the UI remained responsive while the image was being analyzed in the background, providing a much smoother user experience.
